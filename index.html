Object-Oriented Programming (OOP) Concepts

What is OOP and Why Use It in Java?

OOP stands for Object-Oriented Programming, a paradigm that uses objects to design applications. Unlike procedural programming (which focuses on functions), OOP emphasizes objects that interact.

Key Benefits: Modularity (break problems into smaller parts), reusability (use code again without rewriting), and scalability (easier to update large projects).

Java's Role: Java enforces OOP principles strictly—almost everything is an object, and it supports features like garbage collection for memory management.

Insight for Quick Learning: Relate OOP to everyday life. For example, a "Car" isn't just data; it has attributes (color, speed) and actions (drive, brake). This mental model speeds up understanding—always ask: "What real thing am I modeling?"

Core Building Blocks: Classes and Objects

Classes and Objects Explained

Class: A blueprint or template for creating objects. It defines what data (attributes) and behaviors (methods) an object will have.

Object: An instance of a class. It's like creating multiple cars from one car blueprint.

// Class definition class Dog { // Attributes (instance variables) String name; int age; // Method (behavior) void bark() { System.out.println(name + " says: Woof!"); } } public class Main { public static void main(String[] args) { // Creating objects Dog myDog = new Dog(); // Object instantiation myDog.name = "Buddy"; myDog.age = 3; myDog.bark(); // Output: Buddy says: Woof! } }

The Four Pillars of OOP

1. Encapsulation

This "hides" data inside objects and controls access via methods. It protects data from misuse. Use access modifiers like private and getters/setters.

class BankAccount { private double balance; // Encapsulated data public void deposit(double amount) { if (amount > 0) balance += amount; } public double getBalance() { return balance; } }

2. Inheritance

Allows a class to inherit properties and methods from another, promoting reuse. Use the extends keyword.

class Animal { void eat() { System.out.println("Eating..."); } } class Cat extends Animal { // Inheritance void meow() { System.out.println("Meow!"); } } public class Main { public static void main(String[] args) { Cat myCat = new Cat(); myCat.eat(); // Inherited method myCat.meow(); } }

3. Polymorphism

Means "many forms." It lets objects of different classes be treated as one type, often through method overriding or overloading.

class Shape { void draw() { System.out.println("Drawing a shape"); } } class Circle extends Shape { @Override void draw() { System.out.println("Drawing a circle"); } } public class Main { public static void main(String[] args) { Shape s = new Circle(); // Polymorphism s.draw(); // Output: Drawing a circle } }

4. Abstraction

Hides complex details and shows only essentials. Use abstract classes or interfaces.

interface Vehicle { void start(); // Abstract method } class Bike implements Vehicle { public void start() { System.out.println("Bike starting..."); } }

Java Collection Framework

1. ArrayList Methods

add(E e)

Appends an element to the end of the list.

List<String> list = new ArrayList<>(); list.add("A");

add(int index, E element)

Inserts an element at a specific index.

List<String> list = new ArrayList<>(); list.add(0, "B");

addAll(Collection c)

Appends all elements from a collection.

list.addAll(Arrays.asList("C", "D"));

clear()

Removes all elements from the list.

list.clear();

clone()

Returns a shallow copy of the ArrayList.

ArrayList<String> list = new ArrayList<>(); Object cloned = list.clone();

contains(Object o)

Checks if the list contains a specific element.

boolean hasA = list.contains("A");

ensureCapacity(int minCapacity)

Ensures the list has at least a given capacity.

((ArrayList<String>)list).ensureCapacity(100);

get(int index)

Returns the element at a specific index.

String val = list.get(0);

indexOf(Object o)

Returns the index of the first occurrence of an element.

int idx = list.indexOf("A");

isEmpty()

Checks if the list is empty.

boolean empty = list.isEmpty();

remove(int index)

Removes the element at a specific index.

list.remove(0);

sort(Comparator c)

Sorts the list according to the specified comparator.

list.sort(Comparator.naturalOrder());

2. LinkedList Methods

addFirst(E e)

Inserts an element at the beginning.

LinkedList<String> list = new LinkedList<>(); list.addFirst("A");

addLast(E e)

Appends an element to the end.

list.addLast("B");

getFirst()

Retrieves, but does not remove, the first element.

String first = list.getFirst();

getLast()

Retrieves, but does not remove, the last element.

String last = list.getLast();

removeFirst()

Removes and returns the first element.

list.removeFirst();

removeLast()

Removes and returns the last element.

list.removeLast();

push(E e)

Pushes an element onto the stack (equivalent to addFirst).

list.push("C");

pop()

Pops an element from the stack (equivalent to removeFirst).

list.pop();

offer(E e)

Adds an element as the tail (last element) of the list.

list.offer("D");

peek()

Retrieves, but does not remove, the head (first element) of the list.

String peeked = list.peek();

poll()

Retrieves and removes the head (first element) of the list.

String polled = list.poll();

3. HashSet Methods

add(E e)

Adds an element if not already present.

HashSet<String> set = new HashSet<>(); set.add("A");

contains(Object o)

Checks if the set contains an element.

boolean hasA = set.contains("A");

remove(Object o)

Removes the specified element.

set.remove("A");

size()

Returns the number of elements in the set.

int setSize = set.size();

iterator()

Returns an iterator over the elements in the set.

Iterator<String> it = set.iterator();

4. TreeSet Methods

first()

Returns the first (lowest) element.

TreeSet<String> set = new TreeSet<>(); set.add("B"); set.add("A"); String first = set.first(); // "A"

last()

Returns the last (highest) element.

String last = set.last(); // "B"

ceiling(E e)

Returns the least element greater than or equal to the given element.

set.add("C"); String ceil = set.ceiling("B"); // "B"

floor(E e)

Returns the greatest element less than or equal to the given element.

String floor = set.floor("B"); // "B"

higher(E e)

Returns the least element strictly greater than the given element.

String higher = set.higher("A"); // "B"

lower(E e)

Returns the greatest element strictly less than the given element.

String lower = set.lower("B"); // "A"

pollFirst()

Retrieves and removes the first (lowest) element.

set.pollFirst();

pollLast()

Retrieves and removes the last (highest) element.

set.pollLast();

5. HashMap Methods

put(K key, V value)

Associates a value with a key.

HashMap<String, Integer> map = new HashMap<>(); map.put("One", 1);

get(Object key)

Retrieves the value for a key.

Integer val = map.get("One");

remove(Object key)

Removes the mapping for a key.

map.remove("One");

containsKey(Object key)

Checks if the map contains a specific key.

boolean hasOne = map.containsKey("One");

keySet()

Returns a Set view of the keys.

Set<String> keys = map.keySet();

values()

Returns a Collection view of the values.

Collection<Integer> values = map.values();

entrySet()

Returns a Set view of the mappings.

Set<Map.Entry<String, Integer>> entries = map.entrySet();

getOrDefault(Object key, V defaultValue)

Returns the value for a key, or a default value if the key is not found.

Integer val = map.getOrDefault("Two", 2);

6. LinkedHashMap Methods

Maintains Insertion Order

Inherits from HashMap, but iterates in the order keys were inserted.

LinkedHashMap<String, Integer> lmap = new LinkedHashMap<>(); lmap.put("Z", 3); lmap.put("A", 1); // Iteration order: "Z", "A"

7. TreeMap Methods

Sorted by Key

Inherits from HashMap, but keeps keys in sorted order.

TreeMap<String, Integer> tmap = new TreeMap<>(); tmap.put("Z", 3); tmap.put("A", 1); // Iteration order: "A", "Z"

firstKey()

Returns the first (lowest) key.

String first = tmap.firstKey(); // "A"

lastKey()

Returns the last (highest) key.

String last = tmap.lastKey(); // "Z"

firstEntry()

Returns a key-value mapping associated with the least key.

Map.Entry<String, Integer> firstEntry = tmap.firstEntry();

lastEntry()

Returns a key-value mapping associated with the greatest key.

Map.Entry<String, Integer> lastEntry = tmap.lastEntry();

pollFirstEntry()

Removes and returns a key-value mapping associated with the least key.

tmap.pollFirstEntry();

pollLastEntry()

Removes and returns a key-value mapping associated with the greatest key.

tmap.pollLastEntry();

8. Queue Methods (LinkedList Implementation)

offer(E e)

Adds an element to the tail of the queue.

Queue<String> queue = new LinkedList<>(); queue.offer("A");

poll()

Retrieves and removes the head of the queue.

String head = queue.poll();

peek()

Retrieves, but does not remove, the head of the queue.

String peeked = queue.peek();

element()

Retrieves, but does not remove, the head of the queue (throws exception if empty).

String element = queue.element();

remove()

Retrieves and removes the head of the queue (throws exception if empty).

String removed = queue.remove();

String Class Methods

Character and Code Point Access

charAt(int index)

Returns the character at the specified index.

String str = "Java"; char ch = str.charAt(1); // ch is 'a'

codePointAt(int index)

Returns the Unicode code point at the specified index.

String str = "Java"; int codePoint = str.codePointAt(1); // codePoint is 97

codePointBefore(int index)

Returns the Unicode code point before the specified index.

String str = "Java"; int codePoint = str.codePointBefore(2); // codePoint is 97

codePointCount(int begin, int end)

Returns the number of Unicode code points in the specified text range.

String str = "Java"; int count = str.codePointCount(0, 4); // count is 4

offsetByCodePoints(int index, int offset)

Returns the index offset by `codePointOffset` code points.

String str = "Java"; int newIndex = str.offsetByCodePoints(1, 2); // newIndex is 3

Comparison

compareTo(String anotherString)

Compares two strings lexicographically.

String s1 = "A"; String s2 = "B"; int result = s1.compareTo(s2); // result is negative

compareToIgnoreCase(String str)

Compares two strings lexicographically, ignoring case.

String s1 = "a"; String s2 = "A"; int result = s1.compareToIgnoreCase(s2); // result is 0

equals(Object anObject)

Checks if this string is equal to the specified object.

String s1 = "Java"; String s2 = "Java"; boolean isEqual = s1.equals(s2); // isEqual is true

equalsIgnoreCase(String anotherString)

Checks if this string equals another string, ignoring case.

String s1 = "java"; String s2 = "Java"; boolean isEqual = s1.equalsIgnoreCase(s2); // isEqual is true

contentEquals(CharSequence cs)

Checks if this string equals the specified CharSequence.

String str = "Java"; StringBuilder sb = new StringBuilder("Java"); boolean isEqual = str.contentEquals(sb); // isEqual is true

Searching and Indexing

indexOf(int ch)

Returns the index of the first occurrence of the specified character.

String str = "banana"; int index = str.indexOf('a'); // index is 1

indexOf(String str)

Returns the index of the first occurrence of the specified substring.

String str = "Hello World"; int index = str.indexOf("World"); // index is 6

lastIndexOf(int ch)

Returns the index of the last occurrence of the character.

String str = "banana"; int index = str.lastIndexOf('a'); // index is 5

lastIndexOf(String str)

Returns the last index of the specified substring.

String str = "Hello World Hello"; int index = str.lastIndexOf("Hello"); // index is 12

Substrings and Case Conversion

substring(int beginIndex, int endIndex)

Returns a substring from `beginIndex` to `endIndex - 1`.

String str = "Hello World"; String sub = str.substring(6, 11); // sub is "World"

toLowerCase()

Converts all characters to lowercase.

String str = "JAVA"; String lower = str.toLowerCase(); // lower is "java"

toUpperCase()

Converts all characters to uppercase.

String str = "java"; String upper = str.toUpperCase(); // upper is "JAVA"

Whitespace and Trimming

trim()

Removes whitespace from both ends.

String str = " Hello "; String trimmed = str.trim(); // trimmed is "Hello"

strip()

Removes all types of leading and trailing whitespace (Java 11+).

String str = " Hello\t "; String stripped = str.strip(); // stripped is "Hello"

stripLeading()

Removes leading whitespace (Java 11+).

String str = " Hello "; String stripped = str.stripLeading(); // stripped is "Hello "

stripTrailing()

Removes trailing whitespace (Java 11+).

String str = " Hello "; String stripped = str.stripTrailing(); // stripped is " Hello"

Joining, Formatting, and Replacing

join(CharSequence d, CharSequence... e)

Joins strings using the given delimiter (static method).

String joined = String.join("-", "a", "b", "c"); // joined is "a-b-c"

format(String format, Object... args)

Returns a formatted string (static method).

String formatted = String.format("Val: %d", 42); // formatted is "Val: 42"

replace(char old, char new)

Replaces all occurrences of a character.

String str = "Java"; String replaced = str.replace('a', 'o'); // replaced is "Jovo"

replaceAll(String regex, String r)

Replaces all substrings matching the regex.

String str = "a1b2c3"; String replaced = str.replaceAll("\\d", "#"); // replaced is "a#b#c#"

split(String regex)

Splits the string around matches of the given regex.

String str = "a,b,c"; String[] parts = str.split(","); // parts is ["a", "b", "c"]

Inspection and Checks

isEmpty()

Returns true if the string's length is zero.

String str = ""; boolean empty = str.isEmpty(); // empty is true

isBlank()

Returns true if empty or contains only whitespace (Java 11+).

String str = " "; boolean blank = str.isBlank(); // blank is true

startsWith(String prefix)

Tests if the string starts with the specified prefix.

String str = "Hello"; boolean starts = str.startsWith("He"); // starts is true

endsWith(String suffix)

Tests if the string ends with the specified suffix.

String str = "Hello"; boolean ends = str.endsWith("lo"); // ends is true

contains(CharSequence s)

Tests if the string contains the specified sequence.

String str = "Hello"; boolean has = str.contains("ell"); // has is true

Conversion and Repetition

toCharArray()

Converts the string to a new character array.

String str = "Java"; char[] chars = str.toCharArray(); // chars is ['J', 'a', 'v', 'a']

valueOf(int i)

Returns the string representation of an argument (static).

String s = String.valueOf(123); // s is "123"

repeat(int count)

Returns a string repeated `count` times (Java 11+).

String str = "a"; String repeated = str.repeat(3); // repeated is "aaa"

Array Methods (java.util.Arrays)

asList(T... a)

Returns a fixed-size list backed by the specified array.

String[] array = {"a", "b", "c"}; List<String> list = Arrays.asList(array); System.out.println(list); // Output: [a, b, c]

binarySearch(array, key)

Searches for a specified element using the binary search algorithm (requires sorted array).

int[] arr = {10, 20, 30, 40, 50}; int index = Arrays.binarySearch(arr, 30); System.out.println(index); // Output: 2

compare(array1, array2)

Lexicographically compares two arrays.

int[] arr1 = {1, 2, 3}; int[] arr2 = {1, 2, 3}; int result = Arrays.compare(arr1, arr2); System.out.println(result); // Output: 0 (arrays are equal)

copyOf(original, newLength)

Copies the specified array, truncating or padding as necessary.

int[] original = {1, 2, 3}; int[] copied = Arrays.copyOf(original, 5); System.out.println(Arrays.toString(copied)); // Output: [1, 2, 3, 0, 0]

copyOfRange(original, from, to)

Copies a range of the specified array into a new array.

int[] original = {1, 2, 3, 4, 5}; int[] copied = Arrays.copyOfRange(original, 1, 4); System.out.println(Arrays.toString(copied)); // Output: [2, 3, 4]

deepEquals(a1, a2)

Checks if two multi-dimensional arrays are deeply equal.

String[][] s1 = { {"a"}, {"b"} }; String[][] s2 = { {"a"}, {"b"} }; System.out.println(Arrays.deepEquals(s1, s2)); // Output: true

deepHashCode(Object[] a)

Returns a deep hash code for the array.

String[][] s1 = { {"a"}, {"b"} }; System.out.println(Arrays.deepHashCode(s1)); // Output: 30752

deepToString(Object[] a)

Returns a deep string representation of the array.

String[][] s1 = { {"a", "b"}, {"c"} }; System.out.println(Arrays.deepToString(s1)); // Output: [[a, b], [c]]

equals(array1, array2)

Checks if two arrays are equal.

int[] arr1 = {1, 2, 3}; int[] arr2 = {1, 2, 3}; System.out.println(Arrays.equals(arr1, arr2)); // Output: true

fill(array, value)

Assigns the specified value to each element.

int[] arr = new int[5]; Arrays.fill(arr, 10); System.out.println(Arrays.toString(arr)); // Output: [10, 10, 10, 10, 10]

hashCode(array)

Returns a hash code based on the contents of the array.

int[] arr = {1, 2, 3}; System.out.println(Arrays.hashCode(arr)); // Output: 30817

mismatch(array1, array2)

Finds and returns the index of the first mismatch between two arrays.

int[] arr1 = {1, 2, 4}; int[] arr2 = {1, 2, 3}; System.out.println(Arrays.mismatch(arr1, arr2)); // Output: 2

parallelPrefix(array, operator)

Applies a binary operator to each element in the array in parallel.

int[] arr = {1, 2, 3, 4, 5}; Arrays.parallelPrefix(arr, (x, y) -> x + y); System.out.println(Arrays.toString(arr)); // Output: [1, 3, 6, 10, 15]

parallelSetAll(array, generator)

Sets all elements in parallel using a generated function.

int[] arr = new int[5]; Arrays.parallelSetAll(arr, i -> i * 2); System.out.println(Arrays.toString(arr)); // Output: [0, 2, 4, 6, 8]

parallelSort(array)

Sorts the specified array in parallel.

int[] arr = {5, 2, 8, 1, 9}; Arrays.parallelSort(arr); System.out.println(Arrays.toString(arr)); // Output: [1, 2, 5, 8, 9]

setAll(array, generator)

Sets each element using a generated function.

int[] arr = new int[5]; Arrays.setAll(arr, i -> i * 10); System.out.println(Arrays.toString(arr)); // Output: [0, 10, 20, 30, 40]

sort(array)

Sorts the specified array into ascending order.

int[] arr = {5, 2, 8, 1, 9}; Arrays.sort(arr); System.out.println(Arrays.toString(arr)); // Output: [1, 2, 5, 8, 9]

spliterator(array)

Returns a spliterator over the elements in the array.

int[] arr = {1, 2, 3, 4, 5}; Spliterator.OfInt spliterator = Arrays.spliterator(arr); spliterator.forEachRemaining((n) -> System.out.print(n + " ")); // Output: 1 2 3 4 5

stream(array)

Returns a sequential Stream with the specified array as its source.

int[] arr = {1, 2, 3, 4, 5}; IntStream stream = Arrays.stream(arr); stream.forEach(System.out::print); // Output: 12345

toString(array)

Returns a string representation of the array.

int[] arr = {1, 2, 3}; System.out.println(Arrays.toString(arr)); // Output: [1, 2, 3]

StringBuilder Methods

append(value)

Appends the specified value (string, char, int, etc.) to the builder.

StringBuilder sb = new StringBuilder("Hello"); sb.append(" World"); System.out.println(sb.toString()); // Output: Hello World

insert(int offset, value)

Inserts a value at the specified offset position.

StringBuilder sb = new StringBuilder("HelloWorld"); sb.insert(5, " "); System.out.println(sb.toString()); // Output: Hello World

replace(int start, int end, String str)

Replaces the characters between start and end positions with the provided string.

StringBuilder sb = new StringBuilder("Hello Java"); sb.replace(6, 10, "World"); System.out.println(sb.toString()); // Output: Hello World

delete(int start, int end)

Removes characters between the given start and end indexes.

StringBuilder sb = new StringBuilder("Hello World"); sb.delete(5, 11); System.out.println(sb.toString()); // Output: Hello

deleteCharAt(int index)

Removes the character at the specified index.

StringBuilder sb = new StringBuilder("Hello"); sb.deleteCharAt(1); System.out.println(sb.toString()); // Output: Hllo

reverse()

Reverses the sequence of characters.

StringBuilder sb = new StringBuilder("hello"); sb.reverse(); System.out.println(sb.toString()); // Output: olleh

capacity()

Returns the current capacity of the StringBuilder.

StringBuilder sb = new StringBuilder(); System.out.println(sb.capacity()); // Output: 16

ensureCapacity(int minimumCapacity)

Increases the capacity, if needed, to ensure it can hold a minimum number of characters.

StringBuilder sb = new StringBuilder(); sb.ensureCapacity(20); System.out.println(sb.capacity()); // Output: 34 (or more)

charAt(int index)

Returns the character at the specified index.

StringBuilder sb = new StringBuilder("Hello"); System.out.println(sb.charAt(1)); // Output: e

setCharAt(int index, char ch)

Sets the character at the specified index.

StringBuilder sb = new StringBuilder("Hello"); sb.setCharAt(1, 'a'); System.out.println(sb.toString()); // Output: Hallo

length()

Returns the length of the character sequence.

StringBuilder sb = new StringBuilder("Hello"); System.out.println(sb.length()); // Output: 5

setLength(int newLength)

Sets the new length of the builder.

StringBuilder sb = new StringBuilder("Hello World"); sb.setLength(5); System.out.println(sb.toString()); // Output: Hello

substring(int start)

Returns a substring starting from the specified index.

StringBuilder sb = new StringBuilder("Hello World"); System.out.println(sb.substring(6)); // Output: World

substring(int start, int end)

Returns a substring from start (inclusive) to end (exclusive).

StringBuilder sb = new StringBuilder("Hello World"); System.out.println(sb.substring(0, 5)); // Output: Hello

toString()

Converts the builder object to a String.

StringBuilder sb = new StringBuilder("Hello"); String str = sb.toString(); System.out.println(str); // Output: Hello

trimToSize()

Trims the capacity to the current length.

StringBuilder sb = new StringBuilder(50); sb.append("Hello"); sb.trimToSize(); System.out.println(sb.capacity()); // Output: 5

getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin)

Copies characters from the builder into the destination array.

StringBuilder sb = new StringBuilder("Hello World"); char[] dst = new char[5]; sb.getChars(6, 11, dst, 0); System.out.println(dst); // Output: World

indexOf(String str)

Returns the index of the first occurrence of the specified substring.

StringBuilder sb = new StringBuilder("Hello World"); System.out.println(sb.indexOf("World")); // Output: 6

lastIndexOf(String str)

Returns the last occurrence of the specified substring.

StringBuilder sb = new StringBuilder("Hello World Hello"); System.out.println(sb.lastIndexOf("Hello")); // Output: 12

offsetByCodePoints(int index, int codePointOffset)

Returns the index offset by code points.

StringBuilder sb = new StringBuilder("Java\uD83D\uDE00"); // "Java" + grinning face emoji int index = sb.offsetByCodePoints(1, 2); System.out.println(index); // Output: 3

codePointAt(int index)

Returns the Unicode code point at the specified index.

StringBuilder sb = new StringBuilder("J\uD83D\uDE00"); int codePoint = sb.codePointAt(1); System.out.printf("U+%X", codePoint); // Output: U+1F600

codePointBefore(int index)

Returns the Unicode code point before the specified index.

StringBuilder sb = new StringBuilder("J\uD83D\uDE00"); int codePoint = sb.codePointBefore(3); System.out.printf("U+%X", codePoint); // Output: U+1F600

codePointCount(int beginIndex, int endIndex)

Returns the number of Unicode code points between two indices.

StringBuilder sb = new StringBuilder("Java\uD83D\uDE00"); int count = sb.codePointCount(0, sb.length()); System.out.println(count); // Output: 5

subSequence(int start, int end)

Returns a new character sequence from start (inclusive) to end (exclusive).

StringBuilder sb = new StringBuilder("Hello World"); CharSequence sub = sb.subSequence(0, 5); System.out.println(sub); // Output: Hello

Recursion

Factorial

factorial(n)

Calculates the factorial of a non-negative integer using recursion.

public int factorial(int n) { if (n <= 1) { return 1; } else { return n * factorial(n - 1); } } System.out.println(factorial(5)); // Output: 120
